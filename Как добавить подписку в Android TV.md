# Как добавить подписку в Android TV

Источник: [Telegraph — Как добавить подписку в Android TV](https://telegra.ph/Kak-postavit-Secretum-na-Android-TV-11-11)

Теперь вы можете перенести свою подписку из телефона напрямую в приложение V2RayTun на телевизоре Android TV.

## 1) Установите V2RayTun на телевизор

Можно установить приложение из магазина:

<https://play.google.com/store/apps/details?id=com.v2raytun.android>

Или скачать APK напрямую:

<https://github.com/vladimir-fp/v2client/raw/refs/heads/main/android/v2RayTun_5.18.61.apk>

## 2) Подготовьте импорт на телевизоре

1. Откройте V2RayTun на ТВ.
2. Выберите пункт **«Управление»**.
3. Нажмите **«Импорт из QR-кода»**.
4. На экране телевизора появится QR-код.

## 3) Передайте подписку с телефона

1. Откройте V2RayTun на телефоне.
2. Напротив своей подписки нажмите значок QR-кода.
3. Отсканируйте QR-код, который отображается на телевизоре.
4. На ТВ нажмите кнопку включения.

После этого подписка будет добавлена на Android TV.
