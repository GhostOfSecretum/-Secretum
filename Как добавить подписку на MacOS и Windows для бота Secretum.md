# Как добавить подписку на MacOS и Windows для бота Secretum

Источник: [Telegraph — Как добавить подписку на MacOS и Windows для бота Secretum](https://telegra.ph/Kak-dobavit-podpisku-na-MacOS-02-19)

## Ссылка на Telegram-бота

<https://t.me/wgitzbot>

## Шаги добавления подписки

1. Откройте бот <https://t.me/wgitzbot>, нажмите **«Меню»**, далее **«IT Подписки»**.
2. Выберите нужную подписку и нажмите **«Показать ссылку Подписки»**.
3. Скопируйте ссылку и перейдите в приложение V2RayTun на компьютере.
4. В приложении нажмите **«+»** в правом верхнем углу и выберите **Import from clipboard**.

После этого можно проверять подключение.
